Project Report Summary

This project uses Spring Boot MVC and Java file read/write storage.

Modules

  • Product Management
  • Category Management
  • Supplier Management
  • Stock In Management
  • Stock Out / Sales Management
  • User & Role Management

OOP Concepts

  • Encapsulation: private fields + getters/setters
  • Inheritance: BaseEntity, Person, StockTransaction, AdminUser, StaffUser
  • Polymorphism: displayDetails(), calculateStockEffect(), getAccessLevel()
  • Abstraction: Service interfaces + abstract classes
  • Information Hiding: file logic hidden in repository layer

Storage Files

Data is saved in data/*.txt files. CRUD actions are logged in data/audit-log.txt.
